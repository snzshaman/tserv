#### Дополнительно для магистерского уровня

Рекомендуется реализовать оценку **локального тренда** через наклон линейной аппроксимации на скользящем окне.

Пример идеи:

```python
def rolling_slope(series, window=30):
    slopes = [np.nan] * len(series)
    y = series.to_numpy()

    for i in range(window - 1, len(series)):
        y_window = y[i - window + 1:i + 1]
        x_window = np.arange(window)

        if np.isnan(y_window).any():
            continue

        slope = np.polyfit(x_window, y_window, 1)[0]
        slopes[i] = slope

    return pd.Series(slopes, index=series.index)
```

После этого можно добавить:

```python

```
